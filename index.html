Top recommendation

Start with C1 — derive the indexes.

It attacks the single most-repeated friction in the brain: derived state — rosters, counts, descriptions, the manifest's policy block — hand-authored across shallow interfaces that drift. One generator turns folder-plus-frontmatter into the source and the indexes into projections, which also supplies the structure the phantom/orphan health-checks were already written against. Its pattern generalizes to C6 and de-risks C4.

Cheapest high-impact move if you'd rather bank a quick win first: C2 — arm the ingest seam. The deep module already exists and is fixture-tested; it just needs wiring, and it makes the kernel's raw → skill → canonical promise true.
